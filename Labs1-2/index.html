3. ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

  • GET: recuperar datos
  • Head: recupera los headers
  • Post: enviar una entidad, cambiar datos en el servidor
  • PUT: remplaza las representaciones actuales del recurso de destino
  • PATCH: modificar el recurso
  • DELETE: borra el recurso

Fuente: Métodos de petición HTTP - HTTP | MDN. (2020, October 15). Mozilla.org. https://developer.mozilla.org/es/docs/Web/HTTP/Methods

10. ¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

    Las principales diferencias son:
  • HTML5 es compatible con la sintaxis de HTML4 y XHTML 1.0, por lo que las etiquetas se pueden escribir de varias formas.
  • HTML5 puede definir un conjunto de caracteres a usar, para que así no se vean mal algunos símbolos especiales
  • El uso de "!DOCTYPE html" para indicar la versión, es más simple que en las versiones anteriores
  • Se introducen nuevos atributos como "header", "footer", entre otras.
  • Mejora en las APIS.

Fuente: ¿Cuáles son las diferencias entre HTML4 (XHTML 1.0) y HTML5? – Programación en Internet. (2011, January 17).
